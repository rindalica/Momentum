# F&F 사내 인플루언서 관리 서비스

F&F 브랜드의 인지도를 높이기 위한 마케팅 중 하나인
인플루언서 마케팅을 진행하기 위해 인플루언서 관리 대쉬보드 사이트를 제작.

모든 기능은 전부 직접 구현했으며, 모두 백앤드와 연결하여 진행하였습니다.
